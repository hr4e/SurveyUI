Select Questionnaire and Project combination as a default.

This function displays a list of Questionnaires and Projects for the user to select as a default. "Default" means that the identities of the Questionnaire and Project are saved in Session Data for later retrieval during the life of this browser Session. This is a convenience since other functions assume the default identities set by this function.

A questionnaire may or may not have a Project. The Project makes no difference when editing the questionnaire, however, a Project is required to actually execute the questionnaire.
